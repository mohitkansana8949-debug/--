# **App Name**: QuklyStudy

## Core Features:

- Flashcard Generation: Generate flashcards based on user-provided text or imported documents using AI. The AI tool will identify key concepts and create question-answer pairs.
- Study Mode: Allows users to review flashcards in a sequential manner, marking cards as 'learned' or 'to review'.
- Quiz Mode: Generates a quiz based on the flashcards created, providing immediate feedback.
- Progress Tracking: Tracks user progress, highlighting areas needing more focus.
- Text import: The users can upload PDF or text files.

## Style Guidelines:

- Primary color: Sky Blue (#64B5F6), evokes a sense of calm focus.
- Background color: Light Blue (#E3F2FD), soft and unobtrusive.
- Accent color: Soft green (#A5D6A7), suggesting growth and knowledge.
- Font: 'Inter', sans-serif, for clean readability.
- Use flat, modern icons to represent study modes, progress, and import functions.
- Clean and intuitive layout with clear sectioning.
- Subtle animations and transitions.
